[Latest Version = 0.14]

##[Installation]

```shell
	pip install Fancy_Downloader
```

##[How to use]

```python
	import Fancy_Downloader as dl
	d = dl.Download(url=url, type="serial_chunked", verbose=True) #verbose requeries the Fancy_Progressbar lib
	dl.download(d)
	
```
